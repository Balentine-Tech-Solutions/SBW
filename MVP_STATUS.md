## Project Overview

The SBW (Shoot-By-Wire) CLI Tool is a Python-based command-line application for decoding, decrypting, and visualizing SBW log files generated by embedded firmware. The tool implements a complete host-side processing pipeline for weapons systems telemetry data.

### Architecture
```
Input (.sbw) → Block Parsing → Decryption (AES-GCM) → Decompression (LZ4) 
→ TLV Parsing → Export (CSV/JSON) + Visualization (Plots)
```

## Command Reference

### Basic Decode
```bash
sbw-cli decode input.sbw --out output_dir/
```

### Full Analysis
```bash
sbw-cli decode input.sbw --out analysis/ --csv --json --plots
```

### File Information
```bash
sbw-cli info input.sbw --verbose
```

### Validate File Integrity
```bash
sbw-cli validate input.sbw
```

### With Custom Config
```bash
sbw-cli decode input.sbw --out output/ --config custom.json
```
